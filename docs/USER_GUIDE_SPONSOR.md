## Getting Started

### Creating Your Account

1. **Download the App**: Available on iOS, Android, and web
2. **Sign Up**:
   - Open the app and tap "Sign Up"
   - Enter your email and create a secure password
   - Or use "Sign in with Google" for faster registration
3. **Complete Onboarding**:
   - Select your role as "Sponsor" (or "Both" if you also have a sponsor)
   - Set your sobriety date
   - Add a bio to introduce yourself to sponsees

### Setting Up as a Sponsor

After creating your account:

1. **Complete Your Profile**: A complete profile helps sponsees get to know you
2. **Generate Invite Codes**: Create codes to share with potential sponsees
3. **Set Notification Preferences**: Choose how you want to be notified about sponsee activity

## Managing Sponsees

### Inviting Sponsees

#### Generate an Invite Code

1. **Navigate to Profile**: Tap the "Profile" tab
2. **Select "Invite Codes"**: Access your invite code management
3. **Create New Code**:
   - Tap "Generate New Code"
   - Set expiration date (optional, defaults to 30 days)
   - Copy the code
4. **Share with Sponsee**: Send the code via text, email, or in person

#### Managing Invite Codes

- **View Active Codes**: See all unused codes and their expiration dates
- **Revoke Codes**: Cancel codes you no longer want to use
- **Track Usage**: See which codes were used and by whom

### Accepting Sponsees

When someone uses your invite code:

1. **Receive Notification**: You'll get a notification about the new relationship request
2. **Review Profile**: View the potential sponsee's basic information
3. **Accept or Decline**:
   - Accept to start the sponsor-sponsee relationship
   - Decline if you're at capacity or it's not a good fit
4. **Welcome Message**: Send an introductory message after accepting

### Viewing Sponsee Information

Access detailed information about each sponsee:

1. **Navigate to Manage Tab**: View all your sponsees
2. **Select a Sponsee**: Tap to view their profile
3. **View Details**:
   - Current step progress (1-12)
   - Sobriety date and days sober
   - Bio and personal information
   - Task completion history
   - Message history

**Note**: Sponsee slip-up data is private and not visible to you. Sponsees can choose to share this information with you through messages if they wish.

### Managing Sponsee Relationships

#### Updating Sponsee Progress

1. **Open Sponsee Profile**
2. **Tap "Update Step"**
3. **Select New Step**: Move them to the next step when ready
4. **Add Notes**: Optionally add celebration or encouragement
5. **Confirm**: Sponsee receives notification of progress

#### Ending a Relationship

If a sponsee relationship needs to end:

1. **Open Sponsee Profile**
2. **Tap "Manage Relationship"**
3. **Select "End Relationship"**
4. **Confirm**: This action is permanent
5. **Note**: The sponsee will be notified and can find a new sponsor

## Creating and Managing Tasks

Tasks are the primary way you guide sponsees through their recovery journey.

### Creating a Task

#### Manual Task Creation

1. **Navigate to Manage Tasks Tab**
2. **Tap "Create Task"**
3. **Fill in Task Details**:
   - **Title**: Brief, clear task name (e.g., "Write Fourth Step Inventory")
   - **Description**: Detailed instructions and expectations
   - **Sponsee**: Select which sponsee to assign to
   - **Step Number**: Associate with a step (Step 1-12)
   - **Due Date**: Set a reasonable deadline
4. **Review and Save**
5. **Notification Sent**: Sponsee receives notification of new task

#### Using Task Templates

For common tasks, use templates to save time:

1. **Tap "Create from Template"**
2. **Browse Templates**: View pre-made tasks for each step
3. **Select Template**: Choose the appropriate task
4. **Customize**: Edit any details as needed
5. **Assign**: Select sponsee and set due date
6. **Save**: Task is created and sponsee is notified

### Managing Tasks

#### Viewing All Tasks

1. **Navigate to Manage Tasks Tab**
2. **Filter Options**:
   - By sponsee
   - By status (pending, in-progress, completed)
   - By step number
   - By due date

#### Monitoring Task Progress

- **Pending**: Tasks that haven't been started
- **In Progress**: Tasks sponsees are actively working on
- **Completed**: Finished tasks awaiting your review

#### Reviewing Completed Tasks

1. **Open Completed Task**
2. **Review Work**: Read any notes or submissions from sponsee
3. **Provide Feedback**:
   - Send message with encouragement or guidance
   - Create follow-up task if needed
4. **Archive**: Move to task history

### Task Best Practices

#### Setting Clear Expectations

- **Be Specific**: Clearly describe what you want the sponsee to do
- **Provide Examples**: Show what a completed task looks like
- **Set Realistic Deadlines**: Give enough time but maintain momentum
- **Break Down Large Tasks**: Split complex work into smaller, manageable pieces

#### Task Examples by Step

**Step 1 Tasks**:

- "Read pages 21-26 in the Big Book"
- "Journal about powerlessness over alcohol"
- "List 3 examples of unmanageability in your life"

**Step 4 Tasks**:

- "Complete resentments inventory (people)"
- "Complete resentments inventory (institutions)"
- "Complete fears inventory"
- "Complete sex conduct inventory"

**Step 9 Tasks**:

- "Create your amends list"
- "Make amends to [specific person]"
- "Journal about the experience"

## Monitoring Progress

### Sponsee Progress Dashboard

View comprehensive progress for each sponsee:

1. **Navigate to Sponsee Profile**
2. **View Progress Tab**:
   - Current step
   - Days sober
   - Tasks completed vs. pending
   - Recent activity timeline
   - Completion rate

### Progress Indicators

- **Step Progress**: Visual representation of their journey through the recovery steps
- **Task Completion**: Percentage of completed vs. assigned tasks
- **Engagement**: Frequency of app usage and message responses
- **Sobriety Milestones**: Upcoming anniversaries and important dates

### Identifying Struggles

Watch for signs a sponsee might need extra support:

- **Overdue Tasks**: Multiple tasks past due date
- **Low Engagement**: Not logging in or responding to messages
- **Stalled Progress**: Stuck on same task/step for extended period
- **Communication Gaps**: Not responding to messages

**Action**: Reach out proactively with support and encouragement.

## Communication

### Messaging Sponsees

#### Sending Messages

1. **Navigate to Messages Tab**
2. **Select Sponsee**: Choose conversation
3. **Type Message**: Write your message
4. **Send**: Message is delivered instantly

#### Message Types

**Check-ins**:

- "How are you doing with the Step 4 inventory?"
- "Thinking of you today. How's your sobriety?"

**Guidance**:

- "Let's meet this week to discuss your progress"
- "I noticed you completed that task - great work!"

**Support**:

- "I'm here if you need to talk"
- "Remember, progress not perfection"

**Encouragement**:

- "30 days sober - incredible achievement!"
- "You're doing the hard work. Keep going!"

### Communication Best Practices

#### Response Times

- **Urgent Messages**: Respond within 24 hours
- **Regular Check-ins**: At least weekly contact
- **Task Questions**: Respond promptly to unblock progress

#### Tone

- **Supportive**: Encourage and celebrate progress
- **Honest**: Provide truthful feedback with compassion
- **Respectful**: Honor their journey and autonomy
- **Confidential**: Maintain trust and privacy

#### Setting Boundaries

- **Availability**: Communicate your available hours
- **Emergency Protocol**: Share when to call vs. message
- **Response Time**: Set expectations for message replies
- **Crisis Support**: Direct to professional help when needed (988, AA hotline)

#### Personal Information

- **Name**: Your display name for sponsees
- **Bio**: Share your recovery story and experience
  - Years sober
  - Experience with sponsoring
  - Your approach to the recovery steps
- **Sobriety Date**: Your sobriety anniversary
- **Contact Preferences**: How sponsees can reach you

#### Capacity Management

- **Max Sponsees**: Set a limit on active sponsees
- **Availability Status**: Indicate if accepting new sponsees
- **Temporary Unavailable**: Mark yourself unavailable during life events

#### Notification Settings

Control your notification preferences:

- **New Sponsee Requests**: When someone uses your invite code
- **Task Completions**: When a sponsee completes a task
- **Messages**: When you receive a new message
- **Sponsee Milestones**: Sobriety anniversaries and step completions
- **Overdue Tasks**: When sponsee tasks are overdue

#### Privacy & Security

- **Password**: Change your password regularly
- **Two-Factor Authentication**: Enable for added security (coming soon)
- **Data Export**: Download your data
- **Account Deletion**: Permanently delete your account

### Being Both Sponsor and Sponsee

If you're in recovery and also sponsor others:

1. **Set Role to "Both"**: In profile settings
2. **Connect with Your Sponsor**: Use their invite code
3. **Switch Views**: Toggle between sponsee and sponsor perspectives
4. **Separate Accounts**: Your sponsor doesn't see your sponsee relationships

## Best Practices

### Effective Sponsorship Through the App

#### Initial Setup

1. **Complete your profile**: Help sponsees get to know you
2. **Create your first task**: Start with Step 1 reading
3. **Schedule regular check-ins**: Set weekly message times
4. **Share your approach**: Explain how you work through the steps

#### Ongoing Support

1. **Regular Task Creation**: Keep them progressing
2. **Consistent Communication**: Message at least weekly
3. **Celebrate Milestones**: Acknowledge sobriety anniversaries
4. **Monitor Progress**: Check dashboard regularly
5. **Adjust Pace**: Customize to each sponsee's needs

#### Building Strong Relationships

- **Be Present**: Respond to messages promptly
- **Be Consistent**: Maintain regular contact
- **Be Patient**: Recovery is a process
- **Be Honest**: Give truthful, compassionate feedback
- **Be Supportive**: Celebrate successes, support during struggles

### Working with Multiple Sponsees

#### Organization Strategies

- **Use Task Templates**: Save time on common assignments
- **Set Regular Schedules**: Specific days for each sponsee
- **Batch Similar Activities**: Review all completed tasks together
- **Use Filters**: Organize by urgency and sponsee
- **Take Notes**: Remember details about each sponsee

#### Avoiding Burnout

- **Know Your Limits**: Don't take on more than you can handle
- **Set Boundaries**: Establish clear availability
- **Self-Care**: Maintain your own program
- **Ask for Support**: Reach out to your sponsor
- **Take Breaks**: It's okay to pause accepting new sponsees

### Crisis Situations

The app is not for crisis intervention. If a sponsee is in crisis:

**Immediate Danger**:

- Call 988 (Suicide & Crisis Lifeline)
- Call 911 for medical emergencies
- Direct them to emergency services

**Relapse Concerns**:

- Encourage AA meeting attendance
- Suggest connecting with their support network
- Increase check-in frequency
- Consider suggesting professional counseling

**Remember**: You're a sponsor, not a therapist. Know when to refer to professionals.

## Frequently Asked Questions

### How many sponsees should I have?

There's no set number, but quality matters more than quantity. Many sponsors effectively support 3-5 active sponsees. Consider:

- Your available time
- Your experience level
- Complexity of each case
- Your own recovery stability

### Can I sponsor someone remotely?

Yes! The app is designed to support both local and remote sponsorship. Video calls, phone calls, and messaging can supplement in-person meetings.

### What if a sponsee stops responding?

Try these approaches:

1. Send a caring check-in message
2. Call them if you have their number
3. Wait a few days and reach out again
4. Let them know you're there when they're ready
5. If prolonged, consider ending the relationship to focus on active sponsees

### How do I handle task templates?

Task templates are pre-made assignments for common step work. You can:

- Use them as-is
- Customize before assigning
- Create your own templates
- Share templates with other sponsors (coming soon)

### What if I need to take a break from sponsoring?

1. Update your profile to "Not Accepting New Sponsees"
2. Message current sponsees about your availability
3. Help them find temporary support if needed
4. Set a return date if possible
5. Focus on your own recovery

### How do I see sponsee progress history?

1. Go to Manage tab
2. Select sponsee
3. View "History" tab
4. See timeline of:
   - Tasks completed
   - Steps advanced
   - Sobriety milestones
   - Communication frequency
