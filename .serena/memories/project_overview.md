# Project Overview

## 12-Step Tracker

A React Native/Expo mobile and web application designed to support individuals in AA (Alcoholics Anonymous) recovery programs.

## Purpose

The app facilitates the recovery journey by providing:

1. **Sponsor-Sponsee Relationships**: Connect sponsors with sponsees for guided recovery support
2. **Task Assignments**: Sponsors can assign step-related tasks to their sponsees
3. **Progress Tracking**: Monitor advancement through the 12 steps of recovery
4. **Direct Messaging**: Secure communication between sponsors and sponsees
5. **Sobriety Tracking**: Track sobriety dates and milestones
6. **Relapse Support**: Private relapse tracking and recovery restart functionality
7. **Invite System**: Sponsors can invite sponsees using unique codes

## Target Users

- **Sponsors**: Individuals guiding others through the 12-step program
- **Sponsees**: Individuals working through the 12 steps with sponsor support
- **Dual Role**: Users can be both sponsor and sponsee

## Key Features

- Cross-platform support (iOS, Android, Web)
- Secure authentication with email/password and Google OAuth
- Multi-tenant data security via Row Level Security (RLS)
- Real-time notifications
- Step content and reflection prompts
- Dark/light theme support
